# Architecture

This document describes the high-level architecture of the AI Finance Tracker.

- Backend: Python 3.10 modules under `finance_tracker/`
- UI: Streamlit app entry at `main.py`
- Database: Supabase (Postgres) schema in `database/schema.sql`
- AI: ML modules in `ai_modules/` using scikit-learn and Prophet

Follow README for setup steps.
